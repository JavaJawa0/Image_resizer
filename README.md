# 🖼️ JavaJawa Pic Compressor

A modern, fast, and stylish web application to compress and convert images. Forget complex software—optimize your photos directly in your browser!

### 🚀 [Try it live here!](https://javajawa-pic-reducer.streamlit.app/)

---

## ✨ Features

* **Intelligent Compression:** Reduce file size by up to 90% without noticeable quality loss.
* **HEIC Support:** Upload `.heic` photos from your iPhone; the app converts them to standard `.jpg` automatically.
* **Real-time Statistics:** See exactly how much space you are saving before you even download.
* **Neon Beige UI:** A custom-designed, eye-friendly "Neon Beige" theme for a premium user experience.
* **Privacy Focused:** Images are processed in memory and are never stored on the server.

---

## 🛠️ Tech Stack

* **Language:** [Python](https://www.python.org/)
* **Framework:** [Streamlit](https://streamlit.io/)
* **Image Processing:** [Pillow (PIL)](https://python-pillow.org/) & [Pillow-Heif](https://github.com/casper-hansen/pillow-heif)
